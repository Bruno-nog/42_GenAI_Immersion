Provider: Google AI Studio

1. Como obter uma API Key

1. Acesse https://aistudio.google.com/
2. Faça login com uma conta Google
3. Clique em "Get API key"
4. Clique em "Create API key"
5. Copie a chave gerada para utilizar na API

2. Modelos disponíveis e rate limits

gemini-1.5-flash
- Modelo rápido e barato
- Aproximadamente 15 requests por minuto no free tier

gemini-1.5-pro
- Modelo mais avançado
- Rate limit menor no free tier

gemini-2.0-flash
- Modelo otimizado para velocidade e custo
