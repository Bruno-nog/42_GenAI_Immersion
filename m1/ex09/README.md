# EX01

# Name
Exercício introdutório que declara nome e sobrenome em variáveis e imprime o nome completo no terminal.

## O que é aprendido
- declaração de variáveis de texto
- interpolação de strings com f-string
- saída de dados com `print`

# EX02

# Boolean
Exercício que compara dois números e mostra resultados booleanos para igualdade, maior e menor.

## O que é aprendido
- operadores relacionais (`==`, `>`, `<`)
- interpretação de valores booleanos (`True` e `False`)
- formatação de mensagens para exibir resultados

# EX03

# Lists
Exercício que cria uma lista de frutas e percorre cada item para exibição no terminal.

## O que é aprendido
- criação de listas em Python
- iteração com laço `for`
- acesso sequencial aos elementos de uma coleção

# EX04

# Dicts
Exercício que monta um dicionário de frutas e quantidades e imprime chave e valor em cada linha.

## O que é aprendido
- estrutura de dicionários (`dict`)
- iteração com `items()`
- desempacotamento de chave e valor no `for`

# EX05

# Arguments
Exercício que lê argumentos de linha de comando, valida a entrada e imprime o argumento informado.

## O que é aprendido
- uso de `sys.argv`
- validação de quantidade de argumentos
- encerramento controlado com `sys.exit()`

# EX06

# First Function
Exercício que define uma função de saudação e a executa com um nome recebido pela linha de comando.

## O que é aprendido
- definição de funções com tipagem simples
- separação entre lógica de função e execução principal
- uso de `if __name__ == "__main__"`

# EX07

# Greetings from the Farm
Exercício que usa a biblioteca `cowsay` para exibir uma saudação em formato ASCII a partir de um argumento.

## O que é aprendido
- integração com biblioteca externa
- encapsulamento de comportamento em função
- construção de scripts CLI com saída personalizada

# EX08

# Weather
Exercício que consulta APIs públicas para obter latitude/longitude de uma cidade e exibir a temperatura atual.

## O que é aprendido
- requisições HTTP com `requests`
- leitura e navegação em respostas JSON
- composição de funções para resolver um fluxo completo

# EX09

Exercício focado em documentação do módulo, organizando objetivo e aprendizados de forma clara em Markdown.

## O que é aprendido
- escrita de documentação técnica em Markdown
- organização de conteúdo por estrutura de exercícios
- síntese objetiva de conceitos aprendidos
